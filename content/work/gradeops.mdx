## the problem

Handwritten exam scripts have to be read and scored against a rubric,
answer by answer. AI can handle the first pass — but no grade should go
out without a human seeing it first.

## what i built

A platform that OCRs handwritten exam scripts with NVIDIA Nemotron, scores
each answer against instructor rubrics using Gemini, and routes every
result through a human review queue before grades are released.

## how it works

A LangGraph state machine orchestrates the two-stage AI pipeline with
SQLite checkpointing, so pending reviews survive server restarts. The
backend is FastAPI + PostgreSQL with role-based cookie auth; the React
frontend is built from scratch — instructor and TA dashboards, a
keyboard-friendly review queue, and a landing page with Three.js visuals.

## results

Ongoing — in development with the Coding Club at IIT Guwahati since
